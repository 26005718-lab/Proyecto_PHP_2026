# Proyecto PHP
## Descripción
Es un programa de PHP en el que colocamos un saludo de hola mundo y una imagen creada con IA

## Prerrequisitos (tecnologías a instalar)
Debemos de tener ubuntu en una maquina virtual
Debemos tener visual Studio Code
Debemos instalar el paquete de PHP
Debemos instalar el paquete de apache2
Debemos instalar el paquete de git

## Pasos para su ejecución
Primero debemos de ir a la carpeta de /var/www/html 
Luego debemos de darle permisos a las carpetas de apache2
Luego creamos una carpeta para el proyecto
creamos un archivo .php
colocamos el codigo de php
Luego agregamos una imagen a la carpeta de el proyecto
agregamos el codigo para agregar una imagen
guardamos el proyecto
luego en la web colocamos la IP y debe de salir nuestro mensaje y la imagen
